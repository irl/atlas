Search and Filter

You can search for Tor relays and bridges by using keywords. In particular, Atlas enables you to search for:

  • (partial) nicknames (e.g., “moria”)
  • IP addresses (e.g., “128.31.”)
  • fingerprints (e.g., “9695DFC3”)

It is also possible to combine searches, e.g., “moria 128.31.”. Finally, you can use qualifiers to search for relays in specific countries (e.g., “moria country:us”), with specific contact information (e.g., “contact:arma”), or with specific flags (e.g., “flag:Authority”).

Discover

You are able to get a detailed view of how the relay is configured:

  • Nickname
  • OR and Dir Ports
  • Advertised Bandwidth
  • Contact
  • Exit Policies
  • Uptime
  • Family Members
  • Consensus Weight

Graphs

The historical data is available for relays in graph form:

  • Bytes read/written per second (1m/3m/1y/5y)
  • Consensus weight fraction, guard probability and exist probability (1w/1m/3m/1y/5y)
